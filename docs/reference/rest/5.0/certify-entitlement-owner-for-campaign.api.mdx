The `Authorization` must have `Bearer` followed by `Token`.
Mandatory params:
`verifications`
`camapignId`
`entitlement` 1. `entitlement` 2. `endpoint` 3. `action`
`verifier`
`certkey`
`action(if action param is not passed then entitlement lavel action param will be considered)`
Optional params:
`comments`
