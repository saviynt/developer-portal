Starting with Release v23.9, you can use the createtask API to create remove access tasks for inactive accounts.
The `Authorization` must have `Bearer` followed by `Token`.
Mandatory params:   `assignmenttype` - ENTITLEMENTS,   `tasktype` - DEL,   `username`,   `endpointname`,   `accountname`
Optional params:   `comments`   `source`(Default : WEBSERVICE)
