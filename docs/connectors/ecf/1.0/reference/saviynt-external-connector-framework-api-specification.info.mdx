This page lists the API specification for Saviynt External Connector Framework. You can use "Try it out" option to explore the API specification. "Try it out" will require you to authorize with a static token, so you should first click on Authorize button at the top and provide a value, this is a sample specification and there is no actual token validation, so you can provide any string value for e.g. MyBearerToken and click on Authorize, then Close.
