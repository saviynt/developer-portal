This API consults Application Owner for Campaign at Account and Access Certification Stage in SSM.
The `Authorization` must have `Bearer` followed by `Token`.
Mandatory params:
`campaignname` or `campaignid`
`certificationname` or `certkey`
`newassignee`
`accounts` -  1. `accattkey` 2. `consultassociatedentitlements` (optional)
`entitlements` -  1. `accattkey` 2. `entattkey`
Optional params:
`comments`
`updateuser`
