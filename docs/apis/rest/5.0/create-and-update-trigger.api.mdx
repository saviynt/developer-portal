This API call can be used to create and update a trigger for a particular "jobgroup" in SSM.

The `Authorization` must have `Bearer` followed by `Token`.

Mandatory params:

`jobname`

`triggername`

`jobgroup`

`cronexpression`

Optional params:

`valueMap` - sample for triggerChainJob -  
\{  
"savtriggerorderform": "TESTCONN,WSRETRY",  
"onFailureForm": "Stop"  
\}

To know more on list of parameters that can be passed in valueMap, see [Parameters for Job Trigger APIs](https://docs.saviyntcloud.com/bundle/EIC-Admin-v23x/page/Content/Chapter10-Job-Control-Panel/Adding-a-new-job.htm)
