This API is used to approve/reject entire request.

The `Authorization` must have `Bearer` followed by `Token`. 

Mandatory params: 
`requestid` or `requestkey` (if both are passed requestkey will be considered),
`reqaction`:1/2, 1 - approve, 2 - reject

Optional params:
`approver` (if not passed, the token user is used)
`comments`
