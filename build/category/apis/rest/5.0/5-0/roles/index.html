Roles

📄️ Remove Entitlement From Role

This API removes entitlement from role.

📄️ Add Role

This API adds role to user.

📄️ Get Role Details for user

This API can be used to get the list of all the roles.

📄️ Get FireFighter Role Details

This method returns a `List` of active 'FireFighter' roles in SSM.

📄️ Remove Role

This API removes role from user.
